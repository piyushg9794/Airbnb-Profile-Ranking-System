# Airbnb-Profile-Ranking-System
Ranking Airbnb user profile by web scraping

# To Run 

1) Clone the repository
2) Open terminal and navigate to folder
3) Run - python main.py

Input: enter your user url

Output: Your name
        no. of reviews
        among top x% users
